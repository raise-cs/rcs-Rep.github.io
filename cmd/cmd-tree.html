フォルダ内のファイルをツリー表示で取得する

説明

コマンドプロンプトでディレクトリ構造をツリー形式で表示するには、以下のコマンドを使用します。

tree "C:\Users\YourUsername\Documents"

主なオプション

  • /F: ファイルも表示します。

手順

１．コマンドプロンプトを開く

以下の方法でコマンドプロンプトを開きます。

  • 方法1：Windowsキー + R を押し、「cmd」と入力してEnterを押す
  • 方法2：スタートメニューで「コマンドプロンプト」または「cmd」を検索する

２．コマンドを実行する

tree /F "C:\Users\YourUsername\Documents"

このコマンドを実行すると、指定したフォルダ（この例では Documents フォルダ）内のすべてのファイルがツリー表示されます。

実行結果のサンプル

C:\Users\YourUsername\Documents
├── file1.txt
├── file2.txt
└── Subfolder
    └── file3.txt

応用

結果をファイルに出力する方法

tree /F "C:\Users\YourUsername\Documents" >  "C:\Users\YourUsername\output.txt"

これにより、結果が output.txt というファイルに保存されます。

クリップボードにコピーする方法

tree /F "C:\Users\YourUsername\Documents" | clip
クリップボードにコピーした後、メモ帳などのテキストエディタに貼り付けて内容を確認することができます。
